# AINN SLA - Advanced AI Resume Intelligence

AINN SLA is a premium, AI-powered resume screening and career matching application. It leverages Gemini 2.0 to provide deep insights into professional profiles, including multi-dimensional scoring, project analysis, and actionable career recommendations.

## ✨ Features

- **Deep Resume Analysis**: Extracts hard skills, soft skills, projects, and work experience with high precision.
* **Multimodal Extraction**: Can process "scanned" or image-only PDFs using Gemini's visual intelligence.
- **AI Score Breakdown**: Categorical scoring for Technical, Experience, Education, and Presentation.
- **Smart Job Matching**: Matches candidates against a job catalog with AI-generated reasoning.
- **Actionable Tips**: Provides specific "Tips to Match" for every job recommendation.
- **Premium UI**: Glassmorphic design with real-time feedback and dynamic animations.

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **AI Engine**: [Google Gemini 2.0 (Flash)](https://aistudio.google.com/)
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **PDF Processing**: PDF.js (with Multimodal fallback)

## 🛠️ Getting Started

### 1. Prerequisites
- Node.js 18+ 
- A Gemini API Key from [Google AI Studio](https://aistudio.google.com/app/apikey)

### 2. Installation
```bash
git clone <your-repo-url>
cd ainn_sla
npm install
```

### 3. Environment Setup
Create a `.env.local` file in the root directory:
```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run Locally
```bash
npm run dev
```
